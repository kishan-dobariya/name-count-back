# name-count-back
 It gets counts from book.
 
# Set-up Instruction

### Clone repo
```git clone https://github.com/kishan-dobariya/name-count-back.git```
  
### Navigate to folder
```cd name-count-back```

### Install dependencies
```npm i```

### Start server
```npm start```

Your server should be start on port ```3001``` port
